1 Arduino JSON library
2 ====================
3 
4 [![Build Status](https://travis-ci.org/bblanchon/ArduinoJson.svg?branch=master)](https://travis-ci.org/bblanchon/ArduinoJson) [![Coverage Status](https://img.shields.io/coveralls/bblanchon/ArduinoJson.svg)](https://coveralls.io/r/bblanchon/ArduinoJson?branch=master)
5 
6 *An elegant and efficient JSON library for embedded systems.*
7 
8 It's design to have the most intuitive API, the smallest footprint and works without any allocation on the heap (no malloc).
9 
10 It has been written with Arduino in mind, but it isn't linked to Arduino libraries so you can use this library in any other C++ project.
11 
12 Features
13 --------
14 
15 * JSON decoding
16 * JSON encoding (with optional indentation)
17 * Elegant API, very easy to use
18 * Fixed memory allocation (no malloc)
19 * Small footprint
20 * MIT License

22 Quick start
23 -----------
24 
25 #### Decoding / Parsing
26 
27  char json[] = "{\"sensor\":\"gps\",\"time\":1351824120,\"data\":[48.756080,2.302038]}";
28 
29  StaticJsonBuffer<200> jsonBuffer;
30 
31  JsonObject& root = jsonBuffer.parseObject(json);
32 
33  const char* sensor = root["sensor"];
34  long time = root["time"];
35  double latitude = root["data"][0];
36  double longitude = root["data"][1];
37 
38 #### Encoding / Generating
39 
40  StaticJsonBuffer<200> jsonBuffer;
41 
42  JsonObject& root = jsonBuffer.createObject();
43  root["sensor"] = "gps";
44  root["time"] = 1351824120;
45 
46  JsonArray& data = root.createNestedArray("data");
47  data.add(48.756080, 6); // 6 is the number of decimals to print
48  data.add(2.302038, 6); // if not specified, 2 digits are printed
49 
50  root.printTo(Serial);
51  // This prints:
52  // {"sensor":"gps","time":1351824120,"data":[48.756080,2.302038]}
